# 仮面診断 / MTPA (Mask Type Personalize App)

学校・職場など「人前での自分（仮面）」の振る舞いを、4つの軸 × 16タイプで整理する性格診断サイトです。  
※「内面（本来の自分）」ではなく、社会的な場面での外面的な傾向にフォーカスします。

- 公開サイト: https://masktype.synthera.jp/

## 概要
仮面診断（MTPA）は、社会的な場面での対人スタイルや行動傾向を質問から整理し、4文字コード（例: SFCP）として提示します。  
結果ページでは、タイプの特徴・行動傾向を確認できます。

## 4つの軸
- テンション: Sunny / Moon
- ポジション: Front / Back
- 距離感: Close / Guard
- ワークスタイル: Persistent / Quick

## 主なページ
- トップ: 診断の概要
- 診断: 40問に回答（約5分）
- 結果: タイプ詳細（特徴・行動傾向）
- タイプ一覧: 16タイプを一覧表示
- FAQ / 利用規約 / プライバシーポリシー / 免責事項
